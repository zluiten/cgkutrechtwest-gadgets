Class MediaStorage

Description
Description | Vars (details) | Methods (details)

Utility class to store binary Media in a MySQL database.

This class needs its own 3 column table (mediaId, tag, binaryData). tag can be any string, preferrably the mime type for web applications For reusability reasons this class has no dependencies on other classes.
