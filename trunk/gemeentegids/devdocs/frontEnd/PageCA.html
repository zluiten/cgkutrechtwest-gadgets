Class PageCA

Description

Super-class of every page displayed in tab.

This class is a factory for all other pages. Delegates content creation (header/footer) to PageDelegate. PageDelegate may be replaced by a subclass for different applications of TABR. It is also possible to set the PageDelegate with the Page constructor, this affects then only your pages and leaves original TAB pages as they are. This is useful if one might embed TABR into another web application as module to handle login and authentication.
