Error Handler (global Singleton)

Handles all errors that occur during usage of TAB, decides whether the error is fatal ('db','invArg','retVal','script','noLogin' [redirects to login panel], 'denied','adminLock','install','noFile','version','internal') or not ('nonFatal') and outputs an error page, or saves the error respectively. In case of errors Page will print all error messages before any other output
