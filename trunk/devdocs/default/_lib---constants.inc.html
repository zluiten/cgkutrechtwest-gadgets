DEFINE REDO DB VERSION NUMBER (for upgrade.php). This is the last public release version. Must have an upgrade file.

If DB_VERSION_NO changed *AND* x.y.php of DB_VERSION_NO is not found by upgrade.php it will re-run DB_REDO_VERSION_NO . '.php' to make a developer upgrade. This may lead to a DB error or if the upgrade script is carefully crafted to a sucessful upgrade
