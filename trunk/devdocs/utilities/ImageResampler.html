Class ImageResampler

Description
Description | Vars (details) | Methods (details)

Reusable Object for image scaling. No direct error handling.

  1. loads an image from a file
  2. resampling/croping image to a defined size
  3. returns jpeg data as string

Resamples/crops image file to specified width/height


This method reads the requested file, checks, whether it should  just resample it (make the file at most as big as the size defined in  the options) or if it should crop it in a way that it will fill the whole  space available
